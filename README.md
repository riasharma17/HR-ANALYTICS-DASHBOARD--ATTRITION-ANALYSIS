This repository contains a Power BI dashboard designed to analyze employee attrition using key HR metrics. The dashboard helps organizations understand why employees leave, which departments are most affected, and what factors drive attrition.

The project is part of my Power BI analytics work, created to provide meaningful insights using real-world HR data.
